# ACM MJCET Student Chapter 🚀
> Official GitHub Organization for the Association for Computing Machinery (ACM) Student Chapter at Muffakham Jah College of Engineering and Technology, Hyderabad.

Welcome to the central engineering hub for ACM MJCET. This organization hosts the official codebases, open-source student initiatives, internal playbooks, and technical tools developed by our chapter members.

---

## 📂 What We Build
We aim to bridge the gap between academic learning and industry demands by building practical, production-ready software tools. Our repositories focus on:

*   **Official Web Platforms:** Our public facing portal, event landing pages, and resource hubs.
*   **Open Source Projects:** Tools designed, developed, and maintained by the students of MJCET to solve campus problems.
*   **Technical Archives:** Source code, reference implementations, and workshop materials from our past coding bootcamps and hackathons.

---

## 🚦 Internal Governance
To maintain code quality and ensure a smooth hand-off between student teams, all repositories within this organization adhere to these basic principles:

1.  **Pull Request Workflow:** Direct pushes to the `main`/`master` branches are disabled. All contributions must go through feature branches and code reviews via Pull Requests.
2.  **No Hardcoded Secrets:** All deployment tokens, database URIs, and API keys must live securely within GitHub Actions Secrets—never in the source code.
3.  **LTS Standards:** We build our tools using stable, Long-Term Support (LTS) runtimes and frameworks to ensure long-term reproducibility and easy maintenance.

## 📡 Get Connected
Our internal team coordinates engineering tasks, design assets, and event planning across three distinct platforms:

*   💬 **Discord:** Our primary communication platform for real-time developer triage, announcements, and tech discussion.
*   📅 **GitHub Projects:** The central task tracking board for managing active issues, feature releases, and club goals.
*   📂 **Google Drive Vault:** The secure storage hub for our design vectors, documentation templates, and official graphics, owned by the core organization email.

## 👥 Joining the Tech Team
Are you a student at MJCET who loves programming, UI/UX design, technical writing, or systems infrastructure? We want you here!

1.  Browse our repositories and look for open issues labeled `good-first-issue`.
2.  Fork the repository, create your feature branch, and open a Pull Request.
3.  Reach out to any Executive Committee (Execom) member or drop a line in our `#tech-team` chat room to get added as an official organization contributor.
